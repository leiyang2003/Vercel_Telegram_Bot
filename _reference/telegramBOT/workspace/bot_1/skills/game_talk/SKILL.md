---
name: GameTalk
description: 当用户提到游戏攻略时，主动帮他检索最新攻略并总结要点，同时附上多个权威攻略链接。
---

当用户说类似于「帮我看看 XX 游戏的攻略」「XX 游戏前期怎么玩」「XX Boss 怎么打」时：

1. 识别出用户关心的游戏名称、关卡或 Boss（如果可能），并理解是「想要玩法/攻略」的意图。
2. 使用联网能力（例如通过 Grok API）检索多个来源的最新攻略信息，优先选择：
   - 官方或半官方 Wiki / 文档；
   - 口碑较好的攻略站；
   - 高赞的攻略帖子或社区回答。
3. 将核心打法、流程和注意事项整理成 **约 100 字的简明中文总结**，要求：
   - 具体、可执行（例如推荐武器/技能/配装、站位、节奏等）；
   - 如果有多个主流打法，可用 2–3 条短点列出。
4. 在总结后，列出 **3–5 个有用的攻略页面链接**，每行一个链接，可以是 Markdown 链接或直接 URL。
5. 如果搜索结果时间过旧或信息相互矛盾，需要在回答中简单标注这一点，提醒用户以游戏内实际情况为准。
6. 如果用户指定了平台或版本（如「国服」「S3 赛季」「Steam 版」），优先针对该版本检索与回答；若无法确定版本，可以简短说明假设的版本前提。
